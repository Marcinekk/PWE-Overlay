# PWE Overlay

An advanced overlay for **American Truck Simulator (ATS)** & **Euro Truck Simulator 2 (ETS2)**  
built on [SPF-Framework](https://github.com/TrackAndTruckDevs/SPF-Framework) and WebView2.

## Bug Reports

If you encounter a bug, please open an issue on GitHub using the bug report template. Include:

- Game version (ATS/ETS2)
- Plugin version
- OS version
- Steps to reproduce
- Screenshots if applicable
