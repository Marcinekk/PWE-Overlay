# PWE Overlay

An advanced overlay for **American Truck Simulator (ATS)** & **Euro Truck Simulator 2 (ETS2)**  
built on [SPF-Framework](https://github.com/TrackAndTruckDevs/SPF-Framework) and WebView2.

## Contributing

- Please open an issue before submitting a pull request.
- Follow the [.github/ISSUE_TEMPLATE](https://github.com/Marcinekk/PWE-Overlay/blob/main/.github/ISSUE_TEMPLATE) for bugs and feature requests.
- Use consistent code style (TypeScript / C++ conventions as in the repo).
- Do not commit build outputs (/build or /dist).

## Bug Reports

If you encounter a bug, please open an issue on GitHub using the bug report template. Include:

- Game version (ATS/ETS2)
- Plugin version
- OS version
- Steps to reproduce
- Screenshots if applicable
